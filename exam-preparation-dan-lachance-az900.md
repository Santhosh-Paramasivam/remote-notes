# Microsoft Azure Fundamentals: AZ-900 Exam Preparation

- By Dan Lachance

## Knowledge expected

- Understanding of basic cloud concepts
- Understanding of a wide variety of Azure services, and tools for utilizing them

- Knowing when and how to deploy said services

## AZ-900 Exam Domains

- Describe cloud concepts (25 - 30%)

- Describe Azure architecture (availability zones, regions, storage, computing services) and services (35-40%)

- Describe Azure management (knowing what to use, The Azure portal, the CLI, the Powershell) and governance (enabling MFA, conditional access policies, control auth, RBAC) (30-35%)

- Microsoft puts 50 free assessment questions for the AZ-900. Most of the exam is MCQ.

- Microsoft Exam Sandbox
